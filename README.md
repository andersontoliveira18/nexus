# Nexus Contabilidade Digital 🚀
Website institucional moderno e responsivo desenvolvido para o setor contábil.

## ✨ Funcionalidades
- 5 Páginas completas (Home, Sobre, Produtos, Serviços, Contato).
- Chatbot funcional integrado com JavaScript.
- Sistema de filtros na página de Produtos.
- Layout 100% responsivo (Mobile, Tablet e Desktop).
- Filtro de categorias e Modal de Checkout simulado.

## 🛠️ Tecnologias
- HTML5 / CSS3 (Flexbox e Grid)
- JavaScript Vanilla (Puro)
- Organização de pastas profissional (/css, /js).
